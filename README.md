# 📱 TuristaGram_Adaptativo

Aplicación móvil desarrollada en **SwiftUI** que presenta destinos turísticos de forma interactiva, incorporando un diseño **adaptativo y responsivo** para diferentes dispositivos iOS (iPhone y iPad).

Este proyecto fue desarrollado como parte del **Diplomado en Diseño y Programación de Apps iOS (ING2450)**, enfocado en la implementación de interfaces modernas y adaptativas.

---

## 🚀 Descripción

**TuristaGram_Adaptativo** es una evolución de una aplicación base tipo Instagram, orientada a la visualización de lugares turísticos. La app permite al usuario explorar destinos mediante navegación por deslizamiento, interactuar con contenido visual y utilizar funcionalidades nativas del dispositivo.

El enfoque principal del proyecto es la implementación de un **diseño adaptativo**, capaz de ajustarse dinámicamente a distintos tamaños de pantalla, mejorando la experiencia de usuario.

---

## ✨ Funcionalidades principales

* 📍 Visualización de lugares turísticos (ej: Machu Picchu, Galápagos, Bariloche)
* 🔄 Navegación entre destinos mediante **gestos tipo swipe**
* 🖼️ Acceso a la **galería de imágenes** del dispositivo
* 🗺️ Apertura de ubicaciones en **Apple Maps**
* 😊 Sistema de **calificación interactivo con emojis**
* 📱 Interfaz adaptativa para iPhone y iPad
* 🎨 Diseño basado en contenedores visuales tipo tarjeta

---

## 🛠️ Tecnologías utilizadas

* Swift
* SwiftUI
* Xcode
* PhotosUI
* MapKit

---

## 🧠 Conceptos aplicados

* Diseño adaptativo (Responsive Design)
* Uso de **GeometryReader** para layouts dinámicos
* Contenedores de SwiftUI: `VStack`, `HStack`, `ZStack`
* Manejo de estado con `@State`
* Implementación de **Safe Areas**
* Uso de modificadores de diseño (`padding`, `frame`, `alignment`)
* Componentes reutilizables
